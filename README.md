# 停车场管理系统 java791

## 项目概述

停车场管理系统是一个基于B/S架构的web网站，采用MVC模式设计，实现前后端分离。本项目以Java和Vue为主要技术栈，集成了SpringBoot框架和Mybatis-Plus持久层框架，致力于提供高效、稳定的停车场管理解决方案。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows
- **开发环境**: 
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql 8.0
- **技术栈**: 
  - Java
  - Mysql
  - SpringBoot
  - Mybatis-Plus
  - Vue

## 功能模块

### 系统角色

- **普通用户**: 具备以下操作权限
  - 个人信息管理
  - 预定停车位
  - 缴费信息管理

- **管理员用户**: 在普通用户的基础上，增加以下管理权限
  - 用户信息管理
  - 车辆信息管理
  - 停泊车辆管理
  - 车辆进出管理

## 运行环境

- **后端**: 
  - Java 1.8 或更高版本
  - Mysql 8.0 或更高版本
- **前端**:
  - 现代浏览器

## 目录结构

```
|-- 前端代码目录
|-- 后端代码目录
    |-- src
        |-- main
            |-- java
            |-- resources
            |-- webapp
|-- 数据库文件
|-- 项目文档
```

## 部署步骤

1. 准备开发环境，包括Java、Mysql等。
2. 解压源代码，导入数据库文件到Mysql数据库中。
3. 在IDEA等开发工具中配置项目。
4. 编译并运行后端服务。
5. 在前端代码目录下运行开发服务器，打开前端页面。

## 核心亮点

- **前后端分离**: 前后端分离的开发模式，易于维护和扩展。
- **完善的权限控制**: 针对不同用户角色，实现精细化的权限管理。
- **稳定高效**: 基于成熟的SpringBoot框架，确保系统运行稳定高效。
- **完整源码**: 提供完整的源代码，便于二次开发与定制。
